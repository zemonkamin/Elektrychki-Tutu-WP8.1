# Электрички Туту для Windows Phone 8.1

**Уже реализовано:**
- получение станций при выборе начальной и конечной станций
- получение расписания поездов для станции в выбраном направлении
- получение данных о поезде
- сохранение истории выюора  начальной и конечной станций
